# Crazy Marbles
Supinfo 2rd year project.

## Description
This project is a 3D game based on C++, IrrLichtengine and RakNet technologies.
The game is a remake of the NES game "Marble Madness" : https://www.youtube.com/watch?v=CvlbZwoWMgA
